# GravitySim
A project to showcase how shapes react with gravity.

# Compilation
|Component      | Version|
|:-------------:| -----: |
| Gradle        | 7.3    |
| Java          | 17.03  |

A build script has been added because typing out the commands inside it are a bit annoying. To run that script: 
```bash
chmod +x ./buildAndRun.sh
./buildAndRun.sh
```
